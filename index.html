    ¿Como funciona?

    HTML utiliza la antilambda o los diples (osea los aracteres menor que "" y mayor que") para etiquetar texto, imagenes y otros contenido para mostrarlo en un navegador Web.

    La primera pagina siempre debe de llevar el nombre de "index.hmtl"

    Resumen.

    HTML es el lenguaje estándar para crear y estructurar el contenido en la web. Es un lenguaje de marcado, lo que significa que utiliza etiquetas (tags) para describir diferentes elementos dentro de una página web. Estas etiquetas organizan el contenido en títulos, párrafos, imágenes, enlaces, tablas, formularios, y otros componentes que se ven en un navegador.
